# Rong_Spring_SpringMVC_Mybatis

一、登录url  http://127.0.0.1:8080/Rong_Spring_SpringMVC_Mybatis/login
二、项目结构：
        1、/src/main/java   com.rong.ssm.login.controller 为springmvc 视图控制器文件夹
        2、/src/main/java   com.rong.ssm.login.dao 数据持久层接口
        3、/src/main/java   com.rong.ssm.login.service 与 serviceimpl  业务层
        4、/src/main/java   com.rong.ssm.login.entity   对象类
        4、/src/main/resources/log 控制台日志配置 文件
        5、/src/main/resources/Mapper  mybatis文件夹 对象类映射文件
        6、/src/main/resources/properties  文件夹 数据库配置文件
        7、/src/main/resources/spring   spring文件
      
三、开发规范，已com.rong.ssm.login为例，里面新创建文件夹。开发新模块功能。
